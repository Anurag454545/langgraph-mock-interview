# LangGraph Mock Interview System

This project is a complete AI-powered mock interview system built using **Python (FastAPI)**, **LangGraph**, and a **React + Vite frontend**. It generates interview questions, analyzes answers, and provides feedback such as STAR method checks, confidence scoring, and improvement suggestions.
